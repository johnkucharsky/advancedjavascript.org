# Advanced JavaScript Topic Map

## How to use this map

Use this page as the table of contents for the site. Start with the topics you already use at work, then move into the ones that usually break during interviews: async execution, array behavior, `this`, state updates, and data transformation.

The examples are intentionally small. Read the prompt, predict the result, then compare your reasoning with the implementation. That is closer to how these questions appear in real code reviews and technical interviews than solving isolated puzzles by category.

> This resource complements foundational references like
> [MDN Web Docs](https://developer.mozilla.org/en-US/docs/Web/JavaScript), [JavaScript.info](https://javascript.info/), and [react.dev](https://react.dev).
> Use those for API details; use this site to practice reading and explaining code.

## Content

### Graph Algorithms

Traversal patterns used in dependency graphs, routing, search, and coding interviews.

- [Graph Traversal](/graph-algorithms/graph-traversal) – BFS, DFS, adjacency lists, directed and undirected path checks.

### JS Native

Native language behavior explained through small custom implementations and edge cases.

- [Map](/js-native/map-ds) – Map, WeakMap, Map.groupBy.
- [Mutating Methods](/js-native/mutating-methods) – concat, fill, pop, push, reverse, shift, unshift, splice.
- [Promises](/js-native/promises) – Promise.all, race, any, allSettled.
- [Read methods](/js-native/read-only-methods) – at, every, find, findIndex, findLast, includes, indexOf, some.
- [Set and Object](/js-native/set-and-object) – Set, Object.groupBy, Object.assign, Array.entries.
- [Transform Methods](/js-native/transform-methods) – filter, flat, flatMap, join, map, reduce, slice.

### Learn JS and MDN

Core language topics that are easy to use incorrectly when code becomes asynchronous or stateful.

- [Advanced Patterns](/learn-js-and-mdn/advanced-patterns) – Memoize, Mixins, Promise Chaining, Promise Error Handling, Synchronous vs Asynchronous Errors in Promises, Proxy, Range Iterator, Shooters: Closures & Lexical Scope.
- [Delay, Throttling and Debounce](/learn-js-and-mdn/async-and-timers) – Delay Decorator, Delaying function execution, Output Every Second, setInterval, setTimeout, throttling and debounce decorator.
- [Core Concepts](/learn-js-and-mdn/core-concepts) – `this`, loop behavior, object/map conversion, `bind`, async generators, Fibonacci.
- [Event Loop](/learn-js-and-mdn/event-loop) – microtasks, macrotasks, timers, promises, `async`/`await`, and rendering callbacks.
- [This and Object Methods](/learn-js-and-mdn/object-methods) – how `this` changes across object methods, callbacks, arrows, classes, and prototypes.

### Lodash

Small utility functions that are useful to know even when you use [lodash](https://lodash.com/) in production.

- [Chunk, Compact, Partition](/lodash/array-helpers) – chunk, compact, partition.
- [Difference, Intersection, Union](/lodash/difference-and-intersection) – difference, differenceBy, intersection, union.
- [Object Utilities](/lodash/sets-and-objects) – keyBy, omit, orderBy, pick, curry.

### React

React behavior that is easier to understand from render traces than from definitions.

- [React Rendering](/react/rendering) – usePrevious, children.
- [State Patterns](/react/state) – Normalized State, useReducer ( custom implementation ).

### System Design

Small architecture exercises written in plain JavaScript.

- [Redux and Twitter](/system-design/redux-and-twitter) – reducer-driven state updates and a small feed-ranking model.
- [Queue and Stack](/system-design/stack-and-queue) – FIFO/LIFO structures, async queues, and stack-based checks.

### Theory

High-level principles and mental models.

- [Composition vs Inheritance](/theory/composition-vs-inheritance) - Composition, Inheritance and Function Stack.
- [SOLID in React](/theory/solid-react) - SOLID principles in React.
- [Type Conversions](/theory/type-conversions) - JavaScript coercion rules, equality checks, and operator behavior.

### Various

Practical transformations and utility patterns that do not fit a single category.

- [Dictionary of Nested](/various/dictionary-of-nested) – nested category data converted into dictionaries for fast lookup by `id`.
- [Group List by Quarters](/various/group-list-by-quarters) – Groups monthly financial data by calendar quarters.
- [Transform Lists and Trees](/various/list-to-tree) – List to tree, tree to list, flat nested items.
- [Sorting](/various/sorting) – bubble sort, insertion sort, selection sort.
- [Various Utilities](/various/utils) – clsx, contains duplicate, deep clone, filter by related property, filterMap, innerJoin, Reducer pattern with actions, Retry with fixed delay, Shuffle (Fisher-Yates Algorithm), Topological Sort.
